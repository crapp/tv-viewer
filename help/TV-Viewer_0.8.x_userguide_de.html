1. Systemvoraussetungen

Minimum:
  • Die TV-Karte muss korrekt eingerichtet sein und auf dem System müssen sich ivtv-tune und v4l2-ctl befinden. Diese beiden Tools sind Teil des ivtv Treibers, und meist in dem Paket ivtv-utils enthalten.
  • Tcl / Tk >= 8.5 muss installiert sein. Dieses sollte bei jeder Distribution mitgeliefert werden. Alternativ kann man ActiveTCL verwenden, welches auch sehr einfach zu installieren ist.
  • Der integrierte Videoplayer basiert auf MPlayer. Deshalb muss mindestens MPlayer >= 1.0rc2 installiert werden. Es wird empfohlen eine möglichst aktuelle Version zu installieren.
Empfohlene Zusatzpakete:
  • Das Programm bietet an verschiedenen Stellen eine direkte Verknüpfung auf Internetseiten oder Dateien an. Damit man dieses Feature nutzen kann müssen sich die xdg-utils auf dem System befinden.
  • Setzt man Tcl/Tk < 8.6 ein braucht man das Paket tkimg damit die GUI png Icons verwenden kann. Sollte das Paket nicht installiert sein verwendet TV-Viewer gifs, welche aber eine deutlich geringere Qualität haben. Auf Debian basierten System heißt dieses Paket libtk-img.
  • Man kann TV-Viewer auch mit einer Fernbedienung steuern. Dafür muss LIRC auf dem System, korrekt eingerichtet sein. Siehe auch Punkt 4 Fernbedienung

2. Installation

Wenn es keine fertigen Pakete für deine Distribution gibt, kannst du hier das letzte Release herunterladen. Entpacke die Datei in einem Verzeichnis deiner Wahl. In dem Verzeichnis sollte sich die Datei "tv-viewer_install.sh" befinden.

Öffne ein Terminal und führe die Datei als root aus:

./tv-viewer_install.sh

Dieser Befehl installiert TV-Viewer nach "/usr/local/share/tv-viewer", die Konfiguration befindet sich in "~/.tv-viewer".
Sollte die Installation nicht klappen dann überprüfe bitte ob dein System alle Voraussetzungen erfüllt. Du kannst einen Fehlerbericht auf sourceforge.net erstellen.
Will man das Programm wieder entfernen führt man, ebenfalls als root, folgenden Befehl aus:

> ./tv-viewer_install.sh --uninstall

Das Installationsscript erstellt automatisch eine Verknüpfung im Menü unter "Multimedia/TV".

Nach der Installation sollte man die Einstellungen überprüfen um dann mit dem Sendereditor nach Programmen zu suchen.

3. General Usage


3.1 Hauptfenster

Über die Menubar kann man alle Teile des Programmes finden. So zum Beispiel die Einstellungen, der Sendereditor, die Farbverwaltung ...

Timeshift: Diese Funktion ist in Version 0.8.1a1 deaktiviert. Sie wird ab Version 0.8.1 zur Verfügung stehen.

Aufnahme: Startet den Aufnahme-Assistent.

EPG: Führt das Programm aus welches zum Empfang des Fernsehprogrammes ausgewählt wurde.

Start TV: Fernsehwiedergabe starten/anhalten. Dieser Button wird deaktiviert wenn MPlayer vom Programm nicht erkannt wurde oder wenn eine Aufnahme läuft.


3.2 Einstellungen


3.2.1 Allgemein

Hier werden allgemeine Dinge für TV-Viewer eingestellt.

Sprache: Hier Sprache wählen, falls man mit der automatischen Erkennung nicht zufrieden ist.

Fernsehwiedergabe beim Programmstart: Führt dazu, dass die Fernsehwiedergabe automatisch beim Programmstart gestartet wird.

Newsreader: Aktiviert den Newsreader. Dieser sucht, mit dem eingestellten Intervall (Tage), nach Neuigkeiten über das Projekt auf der Homepage.

Electronic Program Guide: Wähle hier ein Programm, mit welchem du das Fernsehprogramm abrufen willst. Standard ist TV-Browser. Man kann aber auch einfach "firefox http://www.tvspielfilm.de/" nehmen.



3.2.2 Analog

Video Device: Das passende video device auswählen. Wenn man sich nicht sicher bist welches Device man wählen soll, dann kann man "dmesg" zusammen mit "grep" verwenden um zu sehen welche Devices der Treiber anlegt.

Video Standard: Hier wählt man den Video Standard und die Frequenztabelle. Beides hängt davon ab wo man sich befindet. "Video Standard erzwingen" kann nützlich sein wenn der Treiber den falschen Standard auswählt.

Video Eingang: Legt den standard Video Eingang fest. Im Sendereditor kann für jeden Sender der Videoeingang festgelegt werden.
Man kann die Tastenkombinationen "Strg-I" / "Strg-Alt-I" dazu verwenden um den Video Eingang zu ändern.

Videostream: Definiere hier die Videobitrate bzw. Videobitrate (peak), Temporal filter und Hardware Lautstärke. Bitte bedenkt, dass die Videobitrate nie höher sein darf als Peak und Peak nie kleiner als Videobitrate. Der Temporal filter kann das Bild verbessern wenn es verwaschen erscheint. Die Hardware Lautstärke beeinflusst die Lautstärke des Videostreams. Die standard Werte für diese Einstellungen werden empfohlen.


3.2.3 DVB

TV-Viewer wird DVB ab Version 0.9.x unterstützen.


3.2.4 Video

Die hier verfügbaren Einstellungen beziehen sich alle auf MPlayer. Die Tooltips zeigen für alle Optionen grundlegende Informationen an. Eine entsprechende Anleitung kann hier gefunden werden.


3.2.5 Audio

Der Audio Ausgabe Treibe bezieht sich auf MPlayer. Alsa wird empfohlen. Wenn man den Sofware mixer verwendet wird die systemweite Lautstärke nicht beeinflusst. Die Einstellung Audio Kanäle verwendet MPlayer und versucht den Ton in die angegebenen Kanäle zu decodieren.


3.2.6 Radio

Radio wird ab Version 0.8.2 unterstützt.


3.2.7 Interface

Thema: Thema für die Oberfläche von TV-Viewer auswählen.

Tooltips: Schaltet die Tooltips für die entsprechenden Bereiche von TV-Viewer ein oder aus.

Splash Screen: Splash Screen beim Starten des Programms anzeigen oder nicht.

Senderliste: Senderliste nach dem Programmstart anzeigen.

Fenstereigenschaften:

Videofenster:
Videofenster mit dem Hauptfenster logisch verknüpfen: Wenn man diese Option aktiviert bekommt das Videofenster keinen eigenen Eintrag in der Taskbar überschreibt aber auch die Einstellung Videofenster docken.

Videofenster im Vollbild Modus starten.

Ausrichtung Senderliste. Legt fest ob die Senderliste im Vollbild-Modus auf der linken oder rechten Seite angezeigt werden soll.

System Tray:
Videofenster docken: Das aktivieren dieser Option führt dazu, dass das Videofenster ebenfalls in den System Tray gedockt wird.

TV-Viewer nach dem Programmstart docken: Dockt TV-Viewer in den System Tray nach dem Programmstart.

On screen Display. Das On screen Display zeigt die Laustärke, den aktuellen Sender, den Zoomfaktur und den derzeitig ausgewählten Videoeingang an.


3.2.8 Aufnahme

Standardpfad: Legt den Standardpfad für Aufnahmen fest.

Standard Aufnahmedauer: Standard Aufnahmedauer für das Aufzeichnen von Sendungen. Format: hh:mm:ss Beispiel 02:30:00 (Aufnahme dauert 2 Stunden, 30 Minuten und 0 Sekunden.)

Senderwechsel während Aufnahme erlauben: Erlaubt das wechseln des Senders während der Aufnahme.

Scheduler automatisch starten: Startet den Scheduler automatisch als Daemon wenn das Betriebssystem gestartet wird. Das ist wichtig falls man Aufnahmen plant und sollte auf alle Fälle aktiviert werden.


3.2.9 Erweitert

Optionen für Mplayer: Hier kann man zusätzlich komandozeilen Optionen für MPlayer angeben, so wie weitere Video- und Audiofilter definieren. Das Handbuch von MPlayer bietet weitere Infos.

Logging: Hier kann man das Protokollieren von TV-Viewer, MPlayer und Scheduler Meldungen aktivieren. Unter Hilfe findest du die passenden Logviewer. Diese Logfiles sind wichtig wenn du einen Fehlerbericht auf sourceforge.net erstellen willst. Fehler- und Warnmeldungen werden im Logviewer "fett" dargestellt.


3.3 Sendereditor

Der Sendereditor wird dazu verwendet eine Senderliste anzulegen.

Sendersuchlauf: Startet die Sendersuch Oberfläche. Hier kann man auswählen auf welchem Videoeingang gesucht werden soll, ob die gefunden Sender an die vorhandene Liste angehängt werden sollen oder ob der gesamte Frequenzbereich durchsucht werden soll.

Hinzufügen: Fügt einen neuen Sender hinzu. Man muss dabei einen Namen, die Frequenz und den Videoeingang angeben.

Löschen: Löscht den ausgewählten Sender.

Bearbeiten: Den ausgewählten Sender anpassen.

(De)Aktivieren: Aktiviert bzw. deaktiviert den ausgewählten Sender. Deaktivierte Kanäle werden rot markiert.

Hoch / Runter : Ändert die Reihenfolge der Kanäle.

Vorschau: Aktiviert die Fernsehwiedergabe. Damit kann man sehen ob die gewählte Frequenz in Ordnung ist und man findet den entsprechenden Sendernamen.

Man kann auch mehrere Sender mit Hilfe der Shift und Strg Taste auswählen.


3.4 Farbverwaltung

Hier kann man die Werte für Helligkeit, Sättigung, Kontrast und Farbton festlegen. Diese Werte gelten für alle Kanäle.


3.5 Aufnahme-Assistent

Der Aufnahme-Assistent ist eine GUI zur planung und verwaltung von Aufnahmen.

Neue Aufnahme: Eine neue Aufnahme erstellen.

  • Wähle den aufzunehmenden Sender aus der Senderliste.
  • Zeit: Die Uhrzeit angeben an der die Aufnahme beginnen soll. Wenn hier nichts angegeben wird startet die Aufnahme sofort.
  • Datum: Datum angeben an welchem die Aufnahme stattfinden soll. Wenn kein Datum angeben wird, wird das aktuelle Datum verwendet. Format: yyyy-mm-dd Beispiel: 2009-08-20
  • Dauer: Dauer der Aufnahme festlegen. Format: hh:mm:ss Beispiel: 02:30:00
  • Auflösung: Videoauflösung. Wird nichts angeben, wird die standard Auflösung verwendet. In den meisten Fällen sollte es nicht nötig sein diesen Wert zu verändern. Es ist besser hinterher eine Software wie Avidemux zu verwenden, um das Video zu bearbeiten.
  • Ausgabedatei: Ausgabedatei und Pfad angeben. Wird nichts angegeben, wird die Aufnahme im standard Pfad für Aufnahmen gespeichert.
Löschen: Lösche eine oder mehrere Aufnahmen.

Bearbeiten: Eine Aufnahme bearbeiten.

Jedes mal wenn man eine Aufnahme hinzufügt, bearbeitet oder löscht wird der Scheduler angehalten und neu gestartet.  Wenn man eine Aufnahme plant und man den Rechner neu startet wird der Scheduler nicht automatisch gestartet. Dafür muss man die entsprechende Option in den Einstellungen aktivieren.
Der Scheduler beendet sich automatisch wenn es keine Jobs mehr gibt.
Den Status des Schedulers und aktive Aufnahmen sieht man in der Statuszeile. Den Button "Aufnahme anhalten" kann man dazu verwenden um eine Aufnahme vorzeitig zu beenden.

Während eine Aufnähme läuft kann TV-Viewer nur im Dateiwiedergabe Modus betrieben werden. Dabei kann man die Aufnahme anschauen. Dieser Modus kann dann beendet werden wenn es keine aktive Aufnahme mehr gibt indem man die Fernsehwiedergabe startet.

4. Fernbedienung

TV-Viewer kann auch durch eine Fernbedienung gesteuert werden. Die Fernbedienung muss mit Hilfe von LIRC eingerichtet sein.

Zuerst sollte man dafür sorgen, dass beim Systemstart das Programm irexec als Daemon ausgeführt wird. "irexec -d"

Dann legt man sich im Homeverzeichnis, wenn sie nicht bereits exisitiert, die Datei ".lircrc" an.

Mögliche Aktionen für die Fernbedienung:
  • teleview (Fernsehwiedergabe starten/stoppen.)
  • station_up (Einen Sender hoch schalten.)
  • station_down (Einen Sender runter schalten.)
  • station_jump (Zwischen den letzten beiden Sendern hin und herwechseln.)
  • key_[0-9] (Tasten 0 - 9 um Sendernummer zu wählen.)
  • fullscreen (In den Vollbildmodus wechseln, bzw. diesen wieder verlassen.)
  • quit (TV-Viewer beenden.)
  • zoom_incr (Den Zoom vergrößern.)
  • zoom_decr (Den Zoom verkleinern.)
  • zoom_wide (Autozoom 16:9)
  • zoom_stnd (Standard Zoom.)
  • record (Aufnahme des aktuellen Kanals starten.)
  • volume_incr (Lautstärke steigern.)
  • volume_decr (Lautstärke verringern.)
  • mute (Stummschaltung ein/aus.)
Die folgenden Aktionen gelten nur für die Wiedergabe einer Aufnahme:
  • pause (Videowiedergabe pausieren/fortsetzen)
  • stop (Videowiedergabe anhalten - Aufnahme wird fortgesetzt.)
  • start (Videowiedergabe starten.)
  • forward_10s (10 sek. vorspulen.)
  • forward_1m (1 min. vorspulen.)
  • forward_10m (10 min. vorspulen.)
  • forward_end (Zum Ende der Wiedergabe springen.)
  • rewind_10s (10 sek. zurückspringen.)
  • rewind_1m (1 min. zurückspringen.)
  • rewind_10m (10 min. zurückspringen.)
  • rewind_start (Zum Anfang der Wiedergabe springen.)
Hier ein Beispiel für die Konfiguration der ".lircrc":

##################################
begin
      remote = Hauppauge #Name der Fernbedienung
      prog = irexec
      button = TV #Bezeichnung des Buttons von der Fernbedieung, der für diese Funktione genutzt werden soll.
      config = tv-viewer & #Startet TV-Viewer, vergesst nicht das "&" Zeichen.
      mode = tv-viewer #Hiermit kann man einen separaten Modus für TV-Viewer starten. Dadurch werden andere Programme nicht beeinflusst.
      repeat = 0 #Soll die Taste wiederholt werden?
    end

begin tv-viewer #Markiert den Beginn des Modus tv-viewer

begin
      remote = Hauppauge
      prog = irexec
      button = Channel_UP
      config = tv-viewer_lirc station_up #Um einen Sender hoch schalten.
      repeat = 0
    end

begin
      remote = Hauppauge
      prog = irexec
      button = Channel_DOWN
      config =tv-viewer_lirc station_down #Um einen Sender runter schalten.
      repeat = 0
    end

begin
      remote = Hauppauge
      prog = irexec
      button = Back/Exit
      config = tv-viewer_lirc quit #TV-Viewer beenden.
      flags = mode #Den Modus tv-viewer verlassen.
      repeat = 0
    end

end tv-viewer #Markiert das Ende des Modus TV-Viewer.
########################################

Weitere Aktionen findet man in der Liste weiter oben.
